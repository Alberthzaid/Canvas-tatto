# Canvas Tattoo - React Project

Proyecto React con TypeScript y Vite para el sitio web de Canvas Tattoo.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
